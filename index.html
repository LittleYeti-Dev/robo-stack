Yeti Knowledge Systems

Robo Stack

Hybrid AI Development Stack — A cybersecurity-hardened development platform leveraging multiple AI coding agents for accelerated workflows.

Architecture

Developer Workstation → Local AI Coding Agents → Git Repository → CI/CD Pipeline → Cloud Infrastructure (AWS) → Client Application

Core Layers

Local Development

Ubuntu workstation with Kubernetes (K3s/Minikube) for container orchestration.

AI Agent Layer

Copilot, CodeWhisperer, JetBrains AI, TabbyML, OpenDevin — multi-agent coding assistance.

Cloud Infrastructure

AWS EC2 compute, VPC networking, optional EKS cluster. All managed via Terraform IaC.

Security

Least-privilege IAM, encrypted APIs, private VPC, dependency scanning, centralized monitoring.
