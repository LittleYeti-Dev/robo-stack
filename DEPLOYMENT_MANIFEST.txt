SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues & Solutions:

1. SSH Connection Denied
   → Check security group allows your IP
   → Verify SSH key permissions (600)
   → Wait 2-3 minutes for rules to propagate

2. K3s Not Running
   → SSH to instance
   → Check: sudo systemctl status k3s
   → Logs: tail -f /var/log/cloud-init-output.log

3. AWS Credentials Error
   → Run: aws configure
   → Verify: aws sts get-caller-identity

4. Terraform Init Fails
   → Check Terraform version >= 1.5
   → Try: rm -rf .terraform && terraform init

5. Instance Type Not Available
   → Try different region
   → Use different instance type (t3.large)
   → Check account limits
