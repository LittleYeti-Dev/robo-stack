Quick Wins (High Risk, Low Effort)

These 7 items can be completed in a single sprint and eliminate the two critical-risk threats:

  1. T01 fix — Edit 2 lines in security-groups.tf to restrict K3s ports
  2. T05 fix — Apply existing network-policy.yaml file (already written)
  3. T06 fix — Add namespace labels for Pod Security Standards
  4. T10 fix — Pin Actions to SHAs (search-and-replace in workflow files)
  5. T07/T08 fix — Traefik middleware for IP whitelist on dev tools
  6. T11 fix — One line added to CODEOWNERS
  7. T09 fix — Enforce PAT expiry in GitHub settings

Strategic Improvements (High Risk, High Effort)

  1. T02 fix — Terraform S3 backend migration (requires S3 bucket + DynamoDB table + state migration)
  2. T03 fix — K3s etcd encryption-at-rest (requires cluster restart with new config)
  3. T04 fix — External Secrets Operator + Vault integration (E5 roadmap)
  4. Runtime security — Falco or similar for container runtime anomaly detection
  5. Admission control — OPA Gatekeeper for policy enforcement
